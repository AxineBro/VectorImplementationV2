# Реализация класса Vector

Пользовательский шаблонный класс Vector на C++, предоставляющий функциональность динамического массива, аналогичную `std::vector`, с дополнительными математическими операциями и утилитами.

## Возможности

- **Операции с динамическим массивом**: Добавление/удаление элементов, вставка/удаление в конкретных позициях
- **Математические операции**: Поэлементное сложение и умножение векторов
- **Поддержка шаблонов**: Работает с любыми типами данных (int, double, std::string и т.д.)
- **Управление памятью**: Автоматическое изменение размера и управление ёмкостью
- **Перегруженные операторы**: Вывод в поток, индексация, сравнение, арифметические операции

## Обзор класса

Класс `Vector<T>` предоставляет следующие основные функции:

### Конструкторы
- `Vector()` - Конструктор по умолчанию (ёмкость: 10)
- `Vector(size_t initial_size)` - Конструктор с начальным размером
- `Vector(const Vector& other)` - Конструктор копирования

### Основные операции
- `push_back(const T& val)` - Добавить элемент в конец
- `push_front(const T& val)` - Добавить элемент в начало
- `insert(size_t pos, const T& val)` - Вставить элемент в позицию
- `erase(size_t pos)` - Удалить элемент в позиции
- `get_size()` - Получить текущее количество элементов

### Операторы
- `operator[]` - Доступ к элементам по индексу
- `operator==` - Сравнение векторов на равенство
- `operator+` - Поэлементное сложение векторов
- `operator*` - Поэлементное умножение векторов
- `operator<<` - Поддержка вывода в поток

## Примеры использования

### Базовое использование с целыми числами
```cpp
Vector<int> v1;
v1.push_back(1);
v1.push_back(2);
v1.push_back(3);
v1.push_front(0);
v1.insert(2, 10);
v1.erase(1);
std::cout << v1 << std::endl; // Вывод: [0, 10, 3]
```

### Математические операции
```cpp
Vector<int> v4(4);
v4[0] = 1; v4[1] = 2; v4[2] = 3; v4[3] = 4;

Vector<int> v5(4);
v5[0] = 5; v5[1] = 6; v5[2] = 7; v5[3] = 8;

Vector<int> sum = v4 + v5;    // [6, 8, 10, 12]
Vector<int> prod = v4 * v5;   // [5, 12, 21, 32]
```

### Работа с разными типами данных
```cpp
// Векторы с double
Vector<double> dv1(2);
dv1[0] = 1.5; dv1[1] = 2.5;

// Векторы с string
Vector<std::string> sv1;
sv1.push_back("Hello");
sv1.push_back("World");
```

## Требования

- C++11 или новее
- Стандартная библиотека: `<iostream>`, `<algorithm>`, `<cstddef>`

## Структура файлов

- `Vector.h` - Заголовочный файл с реализацией шаблона класса Vector
- `main.cpp` - Примеры использования и демонстрация (как в предоставленном коде)

## Важные заметки

- **Проверка размеров**: Математические операции требуют векторы одинакового размера
- **Эффективность памяти**: Автоматически удваивает ёмкость при необходимости
- **Безопасность исключений**: Базовая обработка ошибок для невалидных операций
- **Глубокое копирование**: Конструктор копирования обеспечивает правильное управление памятью

## Ограничения

- Нет проверки границ для доступа по индексу (аналогично обычным массивам)
- Базовая обработка ошибок через вывод в консоль
- Упрощённая реализация по сравнению с `std::vector`

Данная реализация служит образовательным примером проектирования шаблонных контейнеров и может быть расширена дополнительными функциями при необходимости.
